# Microsoft OAuth

Para autentificarse con una cuenta de microsoft en una aplicación es necesario crear una aplicación en el portal de desarrolladores de microsoft.
En la aplicación debe configurarse la url de redirección a la aplicación.

[Microsoft Developer Portal](https://apps.dev.microsoft.com)

[API de REST de Microsoft](https://docs.microsoft.com/es-es/outlook/rest/javascript-tutorial)

[Microsoft Graph API](https://docs.microsoft.com/es-es/graph/auth-v2-user?view=graph-rest-1.0)

[Permisos OAuth](https://docs.microsoft.com/es-es/graph/permissions-reference?view=graph-rest-1.0#user-permissions)

[Microsoft Graph tester](https://developer.microsoft.com/es-es/graph/graph-explorer?request=users?$select=givenName,surname&method=GET&version=v1.0)

[Microsoft Portal Azure](https://portal.azure.com/)
